🔗 Association Rule Mining

Frequent itemset analysis on news keywords to discover latent co-occurrence patterns (e.g., "Economic Reform" → "Policy Shifts", confidence 85%).

Adjust thresholds and run the algorithm to discover keyword relationships.

🌲 Automated Classification

Random Forest classifier trained on TF-IDF vectors to auto-categorize articles into Politics, Technology, Business, and more.

Click "Train & Evaluate" to run the Random Forest classifier. Takes ~15–30 seconds.

🔵 Clustering & Outlier Detection

Unsupervised K-Means clustering aggregates articles into distinct topical clusters without prior labeling. Small clusters are flagged as potential outliers.

Set K and run clustering to discover topic groups in the corpus.
